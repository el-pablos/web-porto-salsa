# Porto — Tama El Pablo

> Personal portfolio website — [porto.tams.codes](https://porto.tams.codes)

## Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (animasi)

## Sections

- Hero — typing animation, CTA, social links
- About — bio, stats
- Skills — progress bar per kategori (Frontend, Backend, AI & Data, DevOps)
- Projects — 6 featured projects dengan tags
- Contact — form + social links
- Footer

## Getting Started

```bash
npm install
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000)

## Deploy ke Vercel

```bash
npx vercel
```

Atau connect repo ke [vercel.com](https://vercel.com) dan auto-deploy.

## Build

```bash
npm run build
npm start
```
